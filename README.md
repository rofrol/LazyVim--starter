# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

## Troubleshooting

If LazyVim behaves strange, like quiting after entering `:` after updating packages, `rm -rf ~/.local/share/nvim/` and restart LazyVim.
